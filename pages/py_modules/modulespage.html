Some Useful Modules for Python

------------------------------------------------------------------
 1.PyAutoGUI - PyAutoGUI is a cross-platform GUI
automation Python module for human beings.
Used to programmatically control the mouse & keyboard.

INSTALLATION - pip install pyautogui

2.TERMCOLOR - Termcolor is a python module which
can chage text color in terminal of a python program.

INSTALLATION - pip install termcolor

3.PYTHON TURTLE - Python Turtle is a module used for
graphics designing in python.

INSTALLATION - pip install PythonTurtle

4.PPRINT - The pprint module provides a capability to
“pretty-print” arbitrary Python data structures in a
form which can be used as input to the interpreter.

INSTALLATION - pip install pprintpp

5.REMBG - Rembg is a tool to remove images background.

INSTALLATION - pip install rembg
